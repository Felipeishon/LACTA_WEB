# LACTA_WEB

## Descripción
Proyecto web built with Vite, Vue/HTML, TailwindCSS, Firebase.

## Requisitos
- Node.js >= 18
- npm

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## Construcción
```bash
npm run build
```

## Variables de entorno
Copiar `.env.example` a `.env` y completar las variables que empiezan con `VITE_`.

## Estructura del proyecto
- `src/` contiene la lógica del front-end.
- `src/firebase.js` configura Firebase.
- `src/modales.js` gestiona los diálogos modales.
- `src/main.js` punto de entrada.
